# Adult-TM-ATAC
Scripts to reproduce single-cell ATAC-seq analysis of adult mouse trabecular meshwork cells

P60_ATAC_TM_Preprocess.R - data preprocessing, construction of ArchR objects<br/>
P60_ATAC_TM_TrackPlot.R - visualization of marker genes in TM cell types<br/>
P60_ATAC_TM_TFRegulator.R - unbiased analysis and visualization of TF regulators<br/>
P60_ATAC_TM_Supplement.R - script to reproduce supplementary figures
